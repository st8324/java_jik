댓글 등록

1. select.jsp
 - 댓글을 입력할 수 있는 요소를 구성
  - 댓글 입력창 : textarea
  - 댓글 등록 버튼 : button
 - 댓글 등록 버튼 클릭 이벤트 등록
  - 로그인을 체크해서 로그인 안된 경우, 로그인창으로 이동할건지 확인하고, 이동한다고 하면 이동
   - confirm함수
  - 댓글 내용을 확인해서 없으면 내용을 입력하라고 알림창을 띄운후 종료
   - alert함수
  - ajax로 댓글을 등록하도록 요청
   - url : /ajax/comment/insert
   - data : 게시글번호, 댓글 내용
   - 완료 후 서버에서 보낸 데이터를 콘솔에 출력

3. 보드컨트롤러
 - 메소드 추가
  - ajax : @ResponseBody
  - url : /ajax/comment/insert
  - 리턴 : Map<Object, Object>
  - 화면에서 보내준 댓글정보(게시글 번호, 댓글내용을)을 콘솔에서 확인
  - 로그인한 사용자 정보를 콘솔에서 확인
  
4. 보드컨트롤러

  - 등록 여부를 map에 담음
