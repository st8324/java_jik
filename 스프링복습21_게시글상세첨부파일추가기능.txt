첨부파일 - 게시글 상세

1. 보드컨트롤러
 - 보드서비스에게 일을 시킴
  > 매개변수 : 게시글번호
  > 리턴타입 : 첨부파일정보들
  > 기능 : 게시글 번호와 일치하는 첨부파일들을 가져옴
 - 가져온 첨부파일들을 화면에 전송

2. 보드 서비스임플
 - 메소드 추가
  > 매개변수 : 게시글번호
  > 리턴타입 : 첨부파일정보들
  > 기능 : 게시글 번호와 일치하는 첨부파일들을 가져옴
 - 보드 다오에게 일을 시킴
  > 매개변수 : 게시글번호
  > 리턴타입 : 첨부파일정보들
  > 기능 : 게시글 번호와 일치하는 첨부파일들을 가져옴

3. 보드 매퍼
 - 메소드 추가
 - select 태그와 쿼리르 이용하여 첨부파일들을 가져옴
 - resultType : FileVO

4. select.jsp
 - 서버에서 보내준 파일 정보를 이용하여 a태그로 링크를 만듬
  > href : /file/파일명
  > download 속성이용 : 값은 원래파일명

5. servlet-context.xml
 - resources태그를 추가(기존코드 복붙 후 수정)
  > mapping="/file/**"
  > location="첨부파일업로드한경로/"
