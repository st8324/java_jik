- 검색기능
 - 검색어를 입력하기 위한 곳=>검색창

1. list.jsp
 - 검색창을 추가
  - form태그 안에 추가
 - 검색타입을 추가
  - form태그 안에 추가
  - 제목, 작성자
 - 검색 버튼을 추가
 - 검색창, 타입에 name을 지정

2. Criteria
 - 검색어와 검색 타입을 멤버변수로 추가
 - search, searchType
 - 기본 생성자에 기본 검색어는 빈 문자열로, 검색 타입은 all로 설정

3. 보드매퍼(검색어에 맞는 게시글 가져오기)
 - 기존 게시글 가져오는 쿼리문에서 타입에 맞게 검색어가 들어간 게시글들을 가져오도록 수정
 - if태그 활용

- 게시글 리스트에서 검색어에 맞는 게시글이 나오는지 확인 

4. 보드컨트롤러(검색어에 맞는 페이지네이션 맞추기)
 - 서비스에게 시킨 게시글 전체 수를 가져오는 메소드 수정
  - 현재 페이지 정보를 넘겨주고 현재 페이지에 맞는 전체 게시글 수 가져오라고 시킴

5. 보드서비스
 - 게시글 전체수 가져오는 메소드를 수정
  - 현재 페이지 정보를 넘겨주고 현재 페이지에 맞는 전체 게시글 수 가져오라고 시킴
5. 보드서비스임플
 - 게시글 전체수 가져오는 메소드를 수정
  - 페이지 정보 null 체크
  - 현재 페이지 정보를 넘겨주고 현재 페이지에 맞는 전체 게시글 수 가져오라고 시킴

6. 보드 다오 
 - 게시글 전체수 가져오는 메소드를 수정
  - 현재 페이지 정보를 넘겨주고 현재 페이지에 맞는 전체 게시글 수 가져오라고 시킴
7. 보드 매퍼
 - 기존 쿼리에 검색 타입과 검색어에 맞는 전체 게시글 수 가져오는 쿼리로 수정
8. 화면에서 페이지가 맞게 동작하는 확인

9. list.jsp
 - 페이지네이션을 클릭할 때 검색어도 적용되도록 해야함
 - 화면에 검색어와 타입이 유지
