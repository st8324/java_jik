댓글 리스트

1. select.jsp
 - 샘플 데이터를 이용하여 댓글 리스트를 배치
 - 샘플 데이터를 이용하여 페이지네이션을 배치
 - 게시글 리스트를 가져오는 함수를 만듬 => 여러번 호출
  - 게시글 상세로 이동했을 때 호출
  - 댓글 등록했을 때 호출
  - 댓글 삭제됐을 때 호출
  - 댓글 수정했을 때 호출
  - 페이지클릭했을 때 호출
  - 매개변수 : 현재페이지정보 => 객체 { 속성명:값, 속성명:값}
  - ajaxPost를 이용하여 서버에 댓글 리스트를 요청
    - async : false
    - dataObj : 현재페이지 정보
    - url : /ajax/comment/list/게시글번호
    - success : 서버에서 보낸 정보를 콘솔에 출력
