- 추천/ 비추천 버튼

- 추천을 클릭 => 추천이되거나 추천이 취소
- 비추천을 클릭 => 비추천이 되거나 비추천이 취소

- 추천/비추천을 했을 때 => 게시글의 추천/비추천 수가 변경

1. select.jsp
 - 추천/비추천 버튼을 추가
 - 추천/비추천 버튼 클릭 이벤트를 등록
  - 로그인 여부 확인 후 로그인 안되어 있으면 확인창(confirm)으로 로그인 페이지로 이동할건지 물어봄
  - ajax를 이용하여 추천 여부를 확인 후, 알림창을 띄우고, 버튼 색상을 변경
   - url : /check/likes
   - method : post
   - 보낼 데이터 : 게시글 번호, 추천 정보
   - 받을 데이터 : 추천/비추천/추천취소/비추천취소 여부
   - 받은 데이터로 해야할 작업 : 알림창 띄우고, 버튼 색상 조절(3번 작업 완료 후 확인가능)

5. 보드 서비스 임플
 - 메소드 추가
   - 회원아이디, 게시글 번호를 이용하여 추천 정보를 가져옴
   - 추천 정보가 없으면 회원 아이디, 게시글번호, 추천상태를 보드 다오에게 주고 추가하라고 요청 후, 추천상태를 리턴
   - 추천 정보가 있으면 회원 아이디, 게시글번호, 추천상태를 보드 다오에게 주고 수정하라고 요청 후 추천상태를 리턴
      - db에 있는 추천 상태와 화면에서 보낸 추천 상태가 같으면(취소하면) 취소 상태로 수정하고, 10 또는 -10을 리턴
      - db에 있는 추천 상태와 화면에서 보낸 추천 상태가 다르면 화면에서 보낸 추천 상태로 수정하고, 1 또는 -1을 리턴
   - 리턴으로 메소드를 빠져나가전에 보드 다오에게 게시글 번호를 알려주면서 게시글 추천/비추천 수를 수정하라고 요청
